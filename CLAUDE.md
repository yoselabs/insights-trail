# insights-trail

This repo contains daily research briefings.

## When inside `data/<date>/<topic>/`

- Read `briefing.md` first — it has synthesized findings, every source URL, stats, and key quotes
- Use `raw.md` for detailed source data, full quotes, and transcript highlights
- Answer from research data, not general knowledge
- Cite by platform and handle (e.g., "per @handle on X", "r/subreddit")

## When inside `data/<date>/`

- Browse all topic folders to see everything researched that day
- Compare across topics for the same date

## When at repo root

- Compare across dates: `data/<date-1>/<topic>/` through `data/<date-n>/<topic>/`

## Cross-topic and cross-date analysis

- To compare topics on same day: read briefings from `data/<date>/<topic-a>/` and `data/<date>/<topic-b>/`
- To track trends: read briefings from `data/<date-1>/<topic>/` through `data/<date-n>/<topic>/`
- Look for: sentiment shifts, new voices appearing, topics gaining/losing traction
