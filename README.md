# insights-trail

Daily AI research briefings across key industry topics.

## Topics

| Topic | Focus |
|-------|-------|
| **agentic-ai** | AI agents, multi-agent orchestration, MCP, Claude Code, agent frameworks |
| **ai-dev-practice** | AI-assisted development, vibe coding, Cursor, Copilot, RAG, AI in production |
| **ai-leadership** | AI engineering management, CTO strategy, building AI teams, enterprise adoption |
| **agent-skills** | Claude Code skills/plugins, MCP servers, agent tool registries, skill ecosystems |

## Structure

```
data/<YYYY-MM-DD>/<topic>/
├── briefing.md   — synthesized findings, all source URLs, stats
└── raw.md        — detailed source data and quotes
```

## Usage

Browse to any `data/<date>/<topic>/briefing.md` for that day's research synthesis.

Each briefing covers Reddit, X/Twitter, YouTube, Hacker News, TikTok, Instagram, Bluesky, Polymarket, and the web.
