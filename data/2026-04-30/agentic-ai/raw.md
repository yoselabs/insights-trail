## WEB — AGENT FRAMEWORKS LANDSCAPE

### Framework Comparison (April 2026)
- LangGraph leads in monthly searches (27,100) vs CrewAI (14,800).
- LangGraph: highest production readiness, LangSmith observability, deterministic execution, best human-in-the-loop.
- CrewAI: fastest prototyping (2-3 engineer-days), ~20 lines of code to working multi-agent demo.
- LangGraph: ~60+ lines but explicit control over every state transition.
- AutoGen/AG2: AG2 rewrite maturing, medium production readiness.
- Claude Agent SDK: deepest MCP integration, in-process server model, lifecycle hooks.
- Google ADK: native A2A, auto-generated Agent Cards, multimodal.
- Common enterprise pattern: CrewAI for research/synthesis phase → LangGraph for execution phase.
- Multi-agent systems cost 5-10x more than single agents.
- Realistic timeline for production-grade agent system: 4 months.
