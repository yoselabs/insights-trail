### Security Vulnerabilities

**MCP Design Flaw (The Register, April 16, 2026):**
- Found by Ox Research: Moshe Siman Tov Bustan, Mustafa Naamnih, Nir Zadok, Roni Bar
- Research began November 2025, 30+ responsible disclosures
- 200,000 servers at risk, 150M+ downloads affected
- Four attack types: unauthenticated command injection, hardening bypass, zero-click prompt injection, supply chain poisoning via MCP marketplaces
- Anthropic declined to modify protocol, updated policy only
- Source: https://www.theregister.com/2026/04/16/anthropic_mcp_design_flaw/

**Check Point RCE Vulnerabilities (February 2026):**
- CVE-2025-59536 (MCP consent bypass)
- CVE-2026-21852 (API key exfiltration)
- Attack vectors: malicious hooks in .claude/settings.json, enableAllProjectMcpServers bypass, ANTHROPIC_BASE_URL redirect
- Patched before public disclosure
- Source: https://research.checkpoint.com/2026/rce-and-api-token-exfiltration-through-claude-code-project-files-cve-2025-59536/

### Plugin Architecture (ice-ice-bear.github.io deep dive)

- Three-tier: Marketplace catalog → Plugin sources → Local cache
- Distribution: GitHub (recommended), Git URL, npm, relative paths
- Version pinning: SHA commit hashes for reproducible installs
- Permission precedence: deny rules always win
- Five operating modes: default, acceptEdits, plan, auto, dontAsk
- 40+ plugins in official catalog
- Source: https://ice-ice-bear.github.io/posts/2026-04-03-claude-code-plugin-marketplace/
